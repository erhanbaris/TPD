# SÖZLÜK\[\].uzunluk\(\)

_Sözlük_\_'nın uzunluğunu geri döndürür.

```text
sözlüğüm1 = { 'açar_1': "Ne kadar uzun bir yazı?" }
gç::satıryaz(sözlüğüm1["açar_1"].uzunluk())

sözlüğüm2 = { 'açar_1': "Ne kadar uzun bir yazı?"; 
              'açar_2': "İkinci yazı" }
gç::satıryaz(sözlüğüm2["açar_1"].uzunluk())
```
